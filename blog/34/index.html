题目

给你一个 32 位的有符号整数 x ，返回 x 中每位上的数字反转后的结果。

如果反转后整数超过 32 位的有符号整数的范围，就返回 0。 假设环境不允许存储 64 位整数（有符号或无符号）。 示例 1： 输入：x = 123 输出：321 示例 2： 输入：x = -123 输出：-321 示例 3： 输入：x = 120 输出：21 示例 4： 输入：x = 0 输出：0 来源：力扣（LeetCode）

解

class Solution {
    public int reverse(int x) {
      long temp = 0;
      while(x != 0){
        temp = temp * 10 + x % 10;
        x /= 10;
      }
      
      if ((int)temp == temp){
        return (int)temp;
      }
      else {
        return 0;
      }
    }
}
